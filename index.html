glinet-sshkeymanagement

Self-hosted opkg/ipk feed: a GL.iNet admin-UI panel to add/remove the SSH public keys that may log in to the router, designed to survive firmware upgrades. Seven architectures.

README · Source on GitHub

Install

curl -fsSL https://digitalcybersoft.github.io/glinet-sshkeymanagement/setup.sh -o /tmp/setup.sh
sh /tmp/setup.sh --dry-run     # detect arch, change nothing
sh /tmp/setup.sh               # add feed + install

This feed is unsigned and GL opkg has check_signature on, so use --force-signature on opkg update and opkg install. A plain opkg update deletes an unsigned feed's list. Panel appears under System → SSH Keys.

Packages

PackageRole
gl-sdk4-sshkeysBackend: rpc handler, uci key store, boot-time renderer
